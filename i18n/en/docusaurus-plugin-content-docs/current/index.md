# Big Apple Community

Big Apple is an open project exploring low-cost communal living.

We attempt to help ordinary people achieve a stable, dignified, and sustainable life with less monetary expenditure — through shared resources, cooperative labor, transparent finances, and accountable governance.

The English documentation is being prepared. For now, please refer to the [Chinese documentation](/).
