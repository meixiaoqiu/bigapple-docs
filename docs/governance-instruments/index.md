# 治理文书

治理文书是大苹果社区内部用于表达共同决定、长期规则和正式记录的文本入口。

本分区使用“治理文书”作为总称，避免使用 law 一类容易被误解为国家法律或外部法律体系的表述。机器可读 API、JSON Schema、OpenAPI 和示例 payload 仍归入 [技术契约](../technical-contracts/overview.md)。

## 文书类型

- 章程：定义社区的长期基础原则和组织边界。
- 决议：记录一次治理流程确认的具体共同决定。
- 政策：说明某一类事务的长期处理口径。
- 规则：定义可执行、可检查的具体要求。
- 修订案：记录对既有治理文书的正式修改。

## 正式有效版本

只有经过治理流程确认，并保留版本、来源、状态和生效信息的版本，才是大苹果社区正式有效版本。

Fork、翻译、改编、摘录、镜像和再发布版本可以依据许可证传播，但不得冒充大苹果社区正式治理文书、正式决议或正式规则。

## 当前状态

当前暂无已发布的正式治理文书。
