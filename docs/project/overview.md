# 项目概览

Big Apple Live OS 是大苹果社区运行的权威系统，用于承载成员、任务、资源、治理、公开事件、仿真实验和审计留痕等核心业务。

当前文档站先整理公开文档的信息架构。第一批迁移聚焦项目定位、产品规划、路线图、架构边界和运行入口；contracts、API schema、数据库结构和更细的开发手册会在后续批次整理。

- [产品路线图](./roadmap.md)：说明 Live OS 从权威原型到开源和生产化的阶段拆分。

- [运行入口与操作边界](../operations/runtime-boundary.md)：说明 control plane、world runtime 和用户入口的职责边界。
