# 页面说明书清单

本文档记录 Big Apple Live OS 当前实际存在的全部页面，以及页面说明书和自动截图的覆盖状态。

清单内容基于 Live OS 源码（路由、视图、模板）的实际检查，不包含规划中尚未实现的页面。

---

## 清单说明

每个页面记录以下字段：

| 字段 | 说明 |
| --- | --- |
| 页面名称 | 面向读者的名称 |
| 实际路径 | 当前真实 URL |
| 模块 | 所属功能模块 |
| 权限 | 公开、登录、角色权限或管理员 |
| 页面类型 | 列表、详情、表单、仪表盘等 |
| 当前状态 | 已实现、部分实现、演示状态 |
| 说明书状态 | 已完成、待创建、不单独创建 |
| 自动截图状态 | 已配置、待配置、暂不支持 |
| 备注 | 登录、动态数据、稳定性等问题 |

**状态标记说明**：

- **不单独创建**：页面过于简单（如登录页）、为动态 ID 详情页（归属对应列表页说明书）、或为 HTMX 局部片段。
- **暂不支持**：需要登录但尚未提供固定演示账号的页面。
- **待配置**：尚未建立固定演示记录或稳定 URL，因此暂未配置自动截图。未来可通过以下方式支持：seed 固定演示记录、查询演示记录的实际主键、使用稳定 slug 或自然键、由截图准备步骤生成运行时 URL。
